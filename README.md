# Veilex

> Trade without a trace.

Privacy-first, MEV-protected DEX on HashKey Chain.
Commit your swap as a hash. Wait 2 blocks. Reveal and execute.
Bots see nothing. Your trade settles clean.

## Stack
- Next.js 14 (App Router)
- Wagmi v2 + Viem
- RainbowKit
- Tailwind CSS
- Pyth Network oracles
- HashKey Chain (Chain ID 133 testnet / 177 mainnet)

## Contracts
See `../veilex-contracts/` for all smart contracts.

## Getting Started
```bash
npm install
cp .env.example .env.local
npm run dev
```
